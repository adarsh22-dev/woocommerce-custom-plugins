## 📋 POLICIES - SUPER SIMPLE NOW!

### What's Changed:
- ❌ Removed: Page selector
- ❌ Removed: Type dropdown
- ❌ Removed: All complexity
- ✅ Added: Just Title + URL!
- ✅ Added: Optional summary

### Form Fields:
1. **Title** - Name it (Return Policy, Terms, etc.)
2. **URL** - Paste your page URL
3. **Summary** - Optional (what AI should say)
4. **Active** - Checkbox (yes/no)

### How It Works:
```
1. Enter Title: "Return Policy"
2. Enter URL: "https://site.com/returns"
3. Add Summary (optional): "30-day returns..."
4. Check "Active"
5. Click "Add"
6. Done! ✅
```

### Examples:
```
Title: Return Policy
URL: https://yoursite.com/return-policy

Title: Privacy Policy  
URL: https://yoursite.com/privacy

Title: Terms & Conditions
URL: https://yoursite.com/terms

Title: Shipping Info
URL: https://yoursite.com/shipping
```
