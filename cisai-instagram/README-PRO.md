## 🚀 Installation

1. **Upload Plugin**
   - Go to WordPress Admin → Plugins → Add New → Upload Plugin
   - Choose `reelswp-pro.zip`
   - Click "Install Now"
   - Click "Activate"

2. **Start Using**
   - Go to **ReelsWP** in sidebar
   - Click "Add New Story"
   - Upload video or image
   - Add products
   - Customize appearance
   - Publish!

## 📝 Shortcodes

### Basic Feed
```
[reelswp]
```

### With Options
```
[reelswp limit="12" columns="4" layout="grid"]
```

### Story Group
```
[reelswp group="summer-collection"]
```

### UGC Form
```
[reelswp_ugc_form]
```

### Single Story
```
[reelswp_story id="123"]
```
