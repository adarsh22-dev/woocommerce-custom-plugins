## 📦 Installation

### Method 1: WordPress Admin (Recommended)

1. Download `cisai.zip`
2. Go to **WordPress Admin → Plugins → Add New**
3. Click **Upload Plugin**
4. Choose the downloaded file
5. Click **Install Now**
6. Click **Activate**

### Method 2: FTP Upload

1. Extract `cisai.zip`
2. Upload the `cisai` folder to `/wp-content/plugins/`
3. Go to **WordPress Admin → Plugins**
4. Find **Cisai** and click **Activate**

### Method 3: cPanel

1. Log into cPanel
2. Go to **File Manager → wp-content/plugins**
3. Upload `cisai.zip`
4. Extract the archive
5. Activate in WordPress Admin

## 🚀 Quick Start

### Create Your First Story

1. **Navigate to Cisai**
   - Go to WordPress Admin
   - Click **Cisai → Add Reel**

2. **Enter Details**
   - Story title (admin reference)
   - Upload image or video
   - Max file size: 50MB

3. **Add Products (Optional)**
   - Click **Product** tab
   - Select WooCommerce products
   - Products become tappable in story

4. **Add Text Overlay (Optional)**
   - Click **Text** tab
   - Enter text
   - Choose font (50+ options)
   - Set color, size, position
   - Add animation

5. **Add Button (Optional)**
   - Click **Buttons** tab
   - Add interactive button
   - Choose action (Shop, Cart, Link)
   - Customize appearance

6. **Publish**
   - Click **Save** or **Publish**

### Display Stories

**Basic Shortcode:**
```
[cisai_reels]
```

**With Options:**
```
[cisai_reels limit="12" layout="grid" columns="4"]
```

**Specific Group:**
```
[cisai_reels group="new-arrivals"]
```

**UGC Form:**
```
[cisai_ugc_form]
```

## 🔧 Troubleshooting

### Stories Not Displaying

**Check:**
1. Stories are published (not draft)
2. Shortcode syntax correct: `[cisai_reels]`
3. Clear all caches
4. Check browser console for errors
5. Plugin is activated

### Video Not Playing

**Solutions:**
1. Use MP4 format (H.264 codec)
2. Reduce file size (max 50MB)
3. Test in different browser
4. Re-upload video
5. Check server PHP limits

### Slow Loading

**Fix:**
1. Enable lazy loading
2. Compress images
3. Use CDN
4. Enable caching
5. Reduce stories per page (10-20)
6. Optimize database

### Analytics Not Tracking

**Check:**
1. Tracking enabled in settings
2. Disable ad-blockers
3. Test in incognito mode
4. Clear cookies
5. View as logged-out user
